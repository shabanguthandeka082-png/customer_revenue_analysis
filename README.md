# customer_revenue_analysis
Excel project analysing customer revenue by country
Customer Revenue Analysis by Country (Excel)



## 📊 Project Overview



This project analyzes customer spending data to identify top-performing countries and revenue distribution using Excel Pivot Tables.



## 🎯 Objective



To determine which countries contribute the most to total revenue and identify potential growth opportunities.



## 🛠 Tools Used



- Microsoft Excel



- Pivot Tables



- Data Cleaning



## 🔍 Key Insights



- The United States contributes over 30% of total revenue.



- The United Kingdom and India are strong secondary markets.



- Lower-performing countries like Poland and UAE show growth potential.
